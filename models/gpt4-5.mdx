# GPT-4.5

- 推理：无需推理
- GPT-4.5 是在微软 Azure AI 超级计算机上训练
- 从今天开始，ChatGPT Pro 用户可以在网页、移动设备和桌面端的模型选择器中选择 GPT-4.5。我们将在下周开始向 Plus 和 Team 用户推出，然后在接下来的一周向 Enterprise 和 Edu 用户推出。
- GPT-4.5 目前不支持 ChatGPT 中的多模态功能，如语音模式、视频和屏幕共享

1. GPT‑4.5 更新历程
2025 年 2 月 27 日发布
GPT‑4.5（内部代号 “Orion”）作为一个介于 GPT‑4o 与即将推出的 GPT‑5 之间的模型正式发布。OpenAI 宣称该模型在减少不准确信息、改善编程能力以及更自然地理解和执行用户指令方面都有显著提升。
EN.WIKIPEDIA.ORG

后续改进
发布后不久（例如 3 月份），OpenAI 对 GPT‑4.5 进行了细节优化，使其在指令遵循、代码生成和对话协作上变得更加直观和高效。这些更新在内部测试和用户反馈中得到体现，并逐步推向部分用户。

2. 免费用户与 Plus 用户的使用差异
A. 在 ChatGPT 聊天框中的使用
免费用户

模型访问：目前免费用户无法使用 GPT‑4.5。免费版本主要基于 GPT‑3.5 或 GPT‑4o（及其衍生模型），不包括 GPT‑4.5 的最新能力。

功能与体验：因此，免费用户在对话质量、指令理解以及编程协助等方面会与 GPT‑4.5 有明显差距。

Plus 用户

模型访问：Plus 用户可在聊天框中切换使用 GPT‑4.5，但通常会有每周大约 45～50 次的消息限额。这一限制主要是出于计算资源成本的考虑。

功能与体验：Plus 用户能够体验到 GPT‑4.5 提供的更高质量、更准确的回答以及更强的代码生成能力，但在高频率对话场景下需要注意限额问题。
COMMUNITY.OPENAI.COM

B. 在 API 使用方式中的区别
API 用户

访问方式：目前 API 访问仅面向付费用户，因此无论是 GPT‑4.5 还是其他新模型，都只对已订阅的开发者开放。免费用户一般无法通过 API 调用 GPT‑4.5。

调用计费：API 调用是基于使用的输入/输出令牌（tokens）进行计费。对于 GPT‑4.5，开发者可以根据任务需要选择相应的模型版本（例如标准版或经过特别优化的版本），但其费用和速率限制均由 OpenAI 的计费策略决定。

限额与扩展性：与聊天窗口不同，API 方式没有固定的“消息次数”概念，而是基于令牌量，因而在大规模应用场景下更灵活。不同的付费层级（例如标准、专业等）可能会有不同的速率限制和优先级保障。

小结
更新历程：GPT‑4.5 自 2025 年 2 月 27 日发布以来，经过内部和用户反馈不断优化，提升了指令遵循、编程和对话协作等方面的表现。

ChatGPT 聊天框：免费用户无法体验 GPT‑4.5；Plus 用户可使用但每周大约只能发 45～50 条消息。

API 方式：GPT‑4.5 仅面向付费 API 用户，通过令牌计费，调用更灵活，但也需根据具体使用层级注意成本和速率限制。
